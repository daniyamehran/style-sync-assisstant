# Style Sync Assistant

## Problem Statement
Personalised shopping recommendations often lack relevance

## Solution
My app combines visual analysis (user uploads a photo of outfit they like) + text constraints (occasion/budget) to generate highly relevant recommendations using Gemini AI.

## Demo
[Live demo link - add after deploying to Netlify]

## How it works
1. User uploads image → AI extracts colors, patterns, style vibe
2. User adds text → "wedding guest dress under $150"
3. AI combines both → 5 personalized recommendations with explanations

## Tech Stack
- HTML/CSS/JavaScript (vanilla)
- Google Gemini API
- Netlify (deployment)

## Run locally
Just open index.html in any browser
